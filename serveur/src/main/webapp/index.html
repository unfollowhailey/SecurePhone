SecurePhone — Interface serveur

Statut et rappel des services actifs

Services

  • Chat TCP : localhost:8081
  • Audio UDP : localhost:50000
  • Vidéo UDP : localhost:50020

Le serveur peut tourner sans base de données (mode dégradé).

API REST

Les servlets sont exposées via le contexte /securephone.

Exemples :

  • /securephone/api/auth/login
  • /securephone/api/rooms

Déploiement via un conteneur (Tomcat/Jetty).

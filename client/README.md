## SecurePhone Client

Interface Swing pour messagerie + audio/vidéo en temps réel.

### Lancer

Depuis la racine du projet :

- `cd client`
- `mvn clean compile`
- `mvn exec:java -Dexec.mainClass="com.securephone.client.SecurePhoneApp"`

### Configuration

Fichier principal : `client/resources/config.properties`

### Tests

- `./test.sh test`
- `./test.sh audio`
